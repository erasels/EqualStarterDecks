# Equal Starter Decks
This mod standardizes starting decks to always have 4 Strikes, 4 Defends and 2 special cards. You can toggle all features.

It only affects Ironclad and Silent, removing a Strike and Defend from Silent and removing a strike from Ironclad. Additionally it adds a Flex (from StS1) to Ironclad's starting deck.

## Configuration
Simply navigate to the mod settings in the options menu and then click on the gear icon after selecting this mod from the list.
You can toggle the removals and the Flex for Ironclad seperately.

## Installation
Make a folder called mods in the same directory as your sts2.exe `Steam\steamapps\common\Slay the Spire 2` and unzip the contents of the release zip file into that folder that folder. *It should be a directory with a pck, dll and json file.*
